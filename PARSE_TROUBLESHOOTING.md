## 🔍 故障排查步骤

### 1️⃣ **检查文本格式**

确保您的文本符合以下格式：

```
地理位置：[您的描述]
企业员工：[您的描述]
高校学生：[您的描述]
商旅客群：[您的描述]
直接竞品：[您的描述]
潜在竞品：[您的描述]
替代娱乐：[您的描述]
1km以内电竞馆：[数字]家
1-2km电竞馆：[数字]家
2-3km电竞馆：[数字]家
电竞馆备注：[您的描述]
电竞酒店1：[名称],[距离]公里,[房间数]间房
商务酒店1：[名称],[距离]公里,[房间数]间房
```

### 2️⃣ **常见格式错误**

| 错误 | 正确 |
|------|------|
| `地理位置  ：xxx` | `地理位置：xxx` (冒号前不要空格) |
| `企业客户：xxx` | `企业员工：xxx` (关键词要匹配) |
| `学生群体：xxx` | `高校学生：xxx` 或 `学生：xxx` |
| `电竞酒店：XX酒店` | `电竞酒店1：XX酒店,1.2公里,50间房` (需要序号和完整格式) |

### 3️⃣ **使用测试示例**

复制以下完整示例进行测试：

```
地理位置：位于未来科技城核心区，毗邻阿里巴巴西溪园区、杭师大仓前校区

企业员工：周边阿里巴巴、网易、字节跳动等大厂，互联网从业者约5万人

高校学生：杭师大仓前校区距离2公里，学生总数约3万人，电竞社团活跃

商旅客群：西湖国际博览中心1公里，商务出差、会展参会人员密集

直接竞品：3km范围内无电竞酒店，市场完全空白

潜在竞品：2家传统网咖，装修陈旧，设备老化

替代娱乐：电影院2家，KTV 3家，但与电竞客群重叠度低

1km以内电竞馆：5家
1-2km电竞馆：8家
2-3km电竞馆：3家
电竞馆备注：周边电竞馆总数16家，市场需求旺盛但不饱和

电竞酒店1：XX电竞酒店,1.2公里,50间房

商务酒店1：如家酒店,0.8公里,80间房
```

### 4️⃣ **查看浏览器控制台**

1. 按 `F12` 打开开发者工具
2. 切换到"Console"标签
3. 点击解析按钮
4. 查看详细错误信息

您会看到类似这样的日志：
```

解析结果: {geographic_location: "...", ...}
✓ 地理位置已填充
✓ 企业员工已填充
...
```

### 5️⃣ **分段测试**

如果完整文本无法解析，尝试只填写部分字段：

**最小测试（只填地理位置）：**
```
地理位置：阿里巴巴西溪园区旁
```

**逐步添加字段：**
```
地理位置：阿里巴巴西溪园区旁
企业员工：周边大厂员工5万+
```

继续添加，直到找到导致问题的字段。

## 🐛 已知问题

### 问题1：电竞馆数量为0不显示
**原因**: 原代码使用 `if (value)` 判断，0被认为是falsy  
**解决**: 已修复为 `if (value !== undefined)`

### 问题2：酒店列表格式解析失败
**原因**: 
- 缺少序号（如"电竞酒店：xxx"应改为"电竞酒店1：xxx"）
- 没有用逗号分隔三要素
- 距离或房间数包含非数字字符

**解决**:
- 必须有序号：`电竞酒店1：`, `电竞酒店2：`...
- 三要素用逗号分隔：`名称,距离,房间数`
- 单位会自动去除，可以写"1.2公里"或"1.2"

###问题3：字段名不匹配
**原因**: 使用了不支持的字段名  
**支持的字段名**:
- ✅ `地理位置` (固定)
- ✅ `企业员工` / `企业年轻员工` / `企业`
- ✅ `高校学生` / `学生` / `大学生`
- ✅ `商旅客群` / `商务客群` / `商旅`
- ✅ `直接竞品` (固定)
- ✅ `潜在竞品` (固定)
- ✅ `替代娱乐` (固定)
