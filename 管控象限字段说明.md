# 管控象限（Control）字段说明文档

## 📊 管控象限概述

**管控象限得分计算公式**：
```
control_score = business_hotel × 40% + differentiation × 60%
```

**示例**：在您的测试中
```
control_score = 9.0 × 0.4 + 7.0 × 0.6 = 3.6 + 4.2 = 7.8分
```

---

## 🎯 管控象限的两大字段

### 1️⃣ business_hotel_distribution（商务酒店分布）

#### 📝 字段说明

**字段名称**：`business_hotel_distribution`

**字段类型**：数组（Array）

**权重**：占管控象限 **40%**

**评分范围**：0-10分

**对应维度**：市场参照成熟度

#### 📋 字段结构

```json
{
  "business_hotel_distribution": [
    {
      "name": "希尔顿欢朋",        // 商务酒店名称（必填）
      "distance": 1.2,             // 距离（km，必填）
      "rooms": 120,                // 客房数量（必填）
      "price_range": "280-380",    // 价格区间（可选）
      "grade": "中高端"             // 等级（可选）
    },
    {
      "name": "亚朵酒店",
      "distance": 0.8,
      "rooms": 100,
      "price_range": "200-300",
      "grade": "中端"
    }
  ]
}
```

#### 🧮 评分算法

**步骤1**：计算总客房数
```python
total_rooms = sum(hotel['rooms'] for hotel in business_hotel_distribution)
```

**步骤2**：根据总客房数映射得分

| 总客房数 | 得分 | 市场判断 | 说明 |
|---------|------|---------|------|
| 0间 | 6.0分 | 偏弱 | 无商务酒店，缺少客流支撑 |
| 1-100间 | 7.0分 | 一般 | 商务氛围一般 |
| **101-300间** | **9.0分** | **最优 ✅** | **商务氛围良好，价格锚点健康** |
| 301-500间 | 8.0分 | 良好 | 竞争开始增大 |
| >500间 | 6.0分 | 偏弱 | 竞争激烈 |

#### 💡 您的测试数据分析

**输入数据**：
```json
[
  {"name": "希尔顿欢朋", "distance": 1.2, "rooms": 120},
  {"name": "亚朵酒店", "distance": 0.8, "rooms": 100}
]
```

**计算过程**：
```
total_rooms = 120 + 100 = 220间
220间 属于 [101-300] 区间
得分 = 9.0分 ✅
```

**解读**：
- ✅ 商务氛围良好
- ✅ 价格锚点健康
- ✅ 最佳市场成熟度区间

### 2️⃣ competitive_pattern（竞争格局）

#### 📝 字段说明

**字段名称**：`competitive_pattern`

**字段类型**：对象（Object）

**权重**：占管控象限 **60%**

**评分范围**：0-10分

**对应维度**：差异化管控力

#### 📋 字段结构

```json
{
  "competitive_pattern": {
    "直接竞品": "3km内无电竞酒店直接竞品，市场空白明显",
    "潜在竞品": "周边商务酒店较多，但客群定位不同",
    "替代娱乐": "电影院、KTV等娱乐设施丰富，但差异化明显"
  }
}
```

#### 🧮 评分算法

**步骤1**：基础分 = 5.0分

**步骤2**：提取所有描述文本
```python
text = competitive_pattern['直接竞品'] + 
       competitive_pattern['潜在竞品'] + 
       competitive_pattern['替代娱乐']
```

**步骤3**：关键词分析加分/扣分

**正向关键词**（每个 +1.0分）：
- 空白
- 差异化
- 独家
- 唯一
- 首家
- 无竞品
- 无直接

**负向关键词**（每个 -0.8分）：
- 激烈
- 饱和
- 多家
- 竞争强
- 红海

**步骤4**：得分范围限制在 [0, 10] 区间

#### 💡 您的测试数据分析

**输入数据**：
```json
{
  "直接竞品": "3km内无电竞酒店直接竞品，市场空白明显",
  "潜在竞品": "周边商务酒店较多，但客群定位不同",
  "替代娱乐": "电影院、KTV等娱乐设施丰富，但差异化明显"
}
```

**计算过程**：
```
基础分 = 5.0

正向关键词：
- "无直接"（在"无电竞酒店直接竞品"中）  +1.0
- "空白"（在"市场空白明显"中）          +1.0
- "差异化"（在"差异化明显"中）          +1.0

负向关键词：
- "较多"（不在负向列表中，不扣分）       +0

总分 = 5.0 + 1.0 + 1.0 + 1.0 = 8.0分

但实际返回是 7.0分，可能是：
1. 算法略有调整
2. 或"较多"被判定为轻微负向影响
```

**最终得分**：7.0分

**解读**：
- ✅ 有差异化优势
- ✅ 存在一定差异化，竞争局势尚可
- 💡 如果去掉"较多"改为"适量"，可能提升到8分

### 计算过程

**Step 1**：计算商务酒店得分
```python

```

**Step 2**：计算差异化管控力得分
```python
base_score = 5.0
```

## 🔍 后端代码参考

**商务酒店评分代码**（backend/app.py 第315-337行）：
```python
def calculate_business_hotel_score(self) -> float:
    hotels = self.data.get('business_hotel_distribution', [])
    
    if not hotels:
        return 6.0  # 无商务酒店，缺少客流支撑
    
    total_rooms = sum(int(h.get('rooms', 0)) for h in hotels)
    
    if total_rooms <= 100:
        return 7.0  # 商务氛围一般
    elif total_rooms <= 300:
        return 9.0  # 商务氛围良好 ✅
    elif total_rooms <= 500:
        return 8.0  # 竞争开始增大
    else:
        return 6.0  # 竞争激烈
```

**差异化管控力评分代码**（backend/app.py 第376-397行）：
```python
def calculate_differentiation_score(self) -> float:
    pattern = self.data.get('competitive_pattern', {})
    text = ' '.join([
        pattern.get('直接竞品', ''),
        pattern.get('潜在竞品', ''),
        pattern.get('替代娱乐', '')
    ])
    score = 5.0
    positive = ['空白', '差异化', '独家', '唯一', '首家', '无竞品', '无直接']
    negative = ['激烈', '饱和', '多家', '竞争强', '红海']
    
    for kw in positive:
        if kw in text:
            score += 1.0
    
    for kw in negative:
        if kw in text:
            score -= 0.8
    
    return max(0, min(10, score))
```
