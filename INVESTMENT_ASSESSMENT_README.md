# 项目投资评估系统 - 新增功能说明

## 🎉 v4.0 新增功能

### ✨ 投资评估模块

在原有「酒店项目评估」基础上，新增**「项目投资评估」**模块，提供更专业、更全面的投资价值分析。

---

## 📊 核心功能

### 1. 六维度智能评估

| 维度 | 权重 | 说明 |
|------|------|------|
| 📍 地理位置 | 优势24% | 评估项目地理位置优势（产业集聚、交通配套等） |
| 👥 核心客流 | 优势36% | 分析三大核心客群（企业员工、高校学生、商旅客群） |
| 💰 竞争格局 | 风险12% | 评估直接竞品、潜在竞品、替代娱乐情况 |
| 🎮 电竞馆分布 | 风险8% | 统计3km内电竞馆数量和分布密度 |
| 🏨 电竞酒店分布 | 风险16% | 分析周边电竞酒店竞争态势 |
| 🏢 商务酒店分布 | 风险4% | 评估商务酒店对客流的支撑作用 |

### 2. 双维度评估模型

- **优势维度（60%）**：地理位置 + 核心客流
- **风险维度（40%）**：竞争格局 + 竞品分布

**综合得分 = 优势得分 × 0.6 + 风险得分 × 0.4**

### 3. 智能评分算法

#### 地理位置评分
- 基于关键词匹配：阿里巴巴、杭师大、CBD、商圈等
- 高价值关键词：+1.5分/个
- 中等价值关键词：+1分/个

#### 核心客流评分
- 加权平均：企业员工40% + 高校学生35% + 商旅25%
- 根据描述长度和质量评分

#### 电竞馆密度评分
- ≤15家：9.5分（竞争适中）
- 15-25家：6-7.5分
- >25家：2-5分（竞争过度）

#### 电竞酒店竞争评分
- 市场空白：10分
- 竞争强度 = Σ(房间数 × 距离权重)
- 距离越近、规模越大，竞争越激烈

### 4. 价值等级判定

| 得分 | 等级 | 建议 |
|------|------|------|
| 8.5-10 | 🟢 高投资价值 | 强烈推荐投资 |
| 7.0-8.4 | 🔵 较高投资价值 | 推荐投资 |
| 5.5-6.9 | 🟡 中等投资价值 | 谨慎投资 |
| 0-5.4 | 🔴 投资价值偏低 | 不建议投资 |

### 5. 可视化展示

- **综合得分卡片**：直观展示投资价值等级
- **维度得分雷达图**：6个维度得分可视化对比
- **评估结论**：
  - ✅ 项目优势列表
  - ⚠️ 风险提示列表
  - 💡 优化建议列表

## 🚀 快速使用

### 1. 启动服务

```bash
# 方式一：PM2启动（推荐）
pm2 start ecosystem.config.cjs

# 方式二：分别启动
# 后端
cd backend && python3 app.py

# 前端
npx wrangler pages dev . --port 3000
```

### 2. 访问页面

- 打开浏览器访问：http://localhost:3000
- 点击导航栏「投资评估」进入评估模块

### 3. 填写表单

依次填写6大核心字段：
1. **地理位置**：详细描述项目位置优势（必填）
2. **核心客流**：至少填写一项客群信息
3. **竞争格局**：分析直接竞品、潜在竞品、替代娱乐
4. **电竞馆分布**：填写3个距离段的电竞馆数量（必填）
5. **电竞酒店分布**：动态添加周边电竞酒店信息
6. **商务酒店分布**：动态添加周边商务酒店信息

### 4. 查看结果

提交后系统自动生成：
- 综合投资价值得分
- 优势维度 vs 风险维度分数
- 6个维度详细得分
- 雷达图可视化
- 项目优势、风险提示、优化建议

## 🔧 开发说明

### 前端开发
- 表单数据收集：`getAssessmentFormData()`
- 表单验证：`validateAssessmentForm()`
- API调用：`handleAssessmentSubmit()`
- 结果展示：`displayAssessmentResult()`
- 草稿管理：LocalStorage自动保存

### 后端开发
- 数据校验：`validate_number()`, `validate_required_text()`
- 评分计算：`InvestmentEvaluator` 类
- 接口路由：`/api/calculate_score`
- 错误处理：统一JSON错误响应

### 评分规则
- 6个独立评分函数，各自负责一个维度
- 优势维度 = (地理40% + 客流60%) × 60%
- 风险维度 = (竞争30% + 竞品70%) × 40%
- 综合得分 = 优势 + 风险
