# AI数据转换提示词

## 🤖 复制以下提示词给AI使用

---

### 提示词模板：

```
请将以下酒店电竞化改造项目信息转换为JSON格式。

【原始数据】
[在这里粘贴您的项目数据]

【JSON格式要求】
{
  "geographic_location": "地理位置描述",
  "core_customer_flow": {
    "企业年轻员工": "企业客群描述",
    "高校学生": "学生客群描述",
    "商旅与参会客群": "商旅客群描述"
  },
  "competitive_pattern": {
    "直接竞品": "直接竞争对手描述",
    "潜在竞品": "潜在竞争对手描述",
    "替代娱乐": "替代娱乐描述"
  },
  "esports_venue_distribution": {
    "1km以内": 数字,
    "1-2km": 数字,
    "2-3km": 数字,
    "备注": "备注信息"
  },
  "esports_hotel_distribution": [
    {"name": "酒店名", "distance": 距离, "rooms": 房间数}
  ],
  "business_hotel_distribution": [
    {"name": "酒店名", "distance": 距离, "rooms": 房间数}
  ]
}

【转换规则】
1. 字符串用双引号，数字不加引号
2. 中文字段名不要改（"企业年轻员工"等）
3. 距离可以是小数（1.2），房间数必须是整数
4. 空列表用 []
5. 只返回JSON，不要额外说明
```

---

杭师大3万学生，电竞氛围活跃

3km内无电竞酒店
有2家老旧网咖

如家酒店0.8公里80间房

## 🎯 使用流程

1. **复制提示词模板**（上方的完整提示词）
2. **替换【原始数据】部分**为您的项目实际数据
3. **发送给AI**（Claude、GPT-4、文心一言等）
4. **复制AI返回的JSON**
5. **粘贴到评估系统**的JSON格式输入框
6. **点击"解析JSON数据"**
7. **提交评估**

---

## ✅ 验证要点

转换后检查：
- ✅ 所有字符串都是双引号 `""`
- ✅ 数字不加引号：`5` 而不是 `"5"`
- ✅ 中文字段名完全匹配
- ✅ 最后一个元素不加逗号
- ✅ 括号正确配对
