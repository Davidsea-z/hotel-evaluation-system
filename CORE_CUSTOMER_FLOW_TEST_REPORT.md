## 📝 优化建议

### 针对核心客流描述的写作建议

#### ✅ 推荐格式（能获得高分）：
```
[客群名称]+[来源/背景]+[数量规模]+[消费特征]+[需求描述]+[预期流量]

示例（企业员工）：
阿里巴巴及周边互联网大厂年轻员工，收入水平高，游戏需求明确，
消费能力强，追求品质体验和社交场景，预计日均潜在客户200-300人次
```

#### 关键要素：
1. **字符数**：50-100字符最优（获得9分基础分）
2. **正向关键词**：至少包含3-5个（如"密集"、"活跃"、"需求旺盛"、"消费能力强"等）
3. **具体数据**：包含流量预估（"日均200-300人次"）
4. **消费特征**：说明客群的消费能力和需求偏好

#### ❌ 避免（会降低得分）：
- 字符数少于20（得0分）
- 仅20-50字符（只得6分）
- 没有正向关键词（无加分）
- 描述过于简单（"有很多学生"）
