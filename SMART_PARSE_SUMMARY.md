## 📋 支持的解析格式

### 1️⃣ **地理位置**
```
地理位置：位于未来科技城核心区，毗邻阿里巴巴西溪园区...
```

### 2️⃣ **核心客流**（三类）
```
企业员工：周边阿里、网易等大厂，年轻员工5万+...
高校学生：杭师大距离2公里，学生3万人...
商旅客群：西湖会展中心1公里，商务出差频繁...
```

### 3️⃣ **竞争格局**（三维度）
```
直接竞品：3km内无电竞酒店，市场空白
潜在竞品：2家传统网咖，装修陈旧
替代娱乐：电影院2家，KTV 3家
```

### 4️⃣ **电竞馆分布**
```
1km以内电竞馆：5家
1-2km电竞馆：8家
2-3km电竞馆：3家
电竞馆备注：市场需求旺盛但不饱和
```

### 5️⃣ **电竞酒店分布**
```
电竞酒店1：XX电竞酒店,1.2公里,50间房
电竞酒店2：YY主题酒店,2.8公里,30间房
```

### 6️⃣ **商务酒店分布**
```
商务酒店1：如家酒店,0.8公里,80间房
商务酒店2：汉庭酒店,1.5公里,100间房
```

## 🔧 技术实现

### 核心函数
```javascript
parseInvestmentText(text)
  - 输入：用户粘贴的文本
  - 输出：结构化的评估数据对象
  - 功能：正则表达式匹配 + 智能分词

fillFormWithParsedData(data)
  - 输入：解析后的数据对象
  - 输出：填充表单字段
  - 功能：DOM操作 + 动态列表构建
```

### 正则表达式示例
```javascript
// 地理位置匹配
/^地理位置[：:]/i

// 电竞馆数量匹配
/1\s*km\s*以内|1\s*公里以内/i.test(line)
line.match(/(\d+)\s*家/)

// 酒店列表解析
/电竞酒店\d+[：:]/i.test(line)
parts = line.split(/[,，]/)
```
