# 智能文本解析功能 - 使用指南

## 🎯 功能简介

**智能文本解析**功能允许用户直接粘贴包含项目信息的文本，系统自动识别并填充到评估表单中，大幅提升数据录入效率。

---

## ✨ 核心特性

### 1. 一键解析
- ✅ 粘贴文本 → 点击"一键解析" → 自动填充表单
- ✅ 支持中英文冒号（：或:）
- ✅ 自动去除单位（家、公里、间房等）
- ✅ 智能匹配字段名称（支持多种表达方式）

### 2. 支持的字段格式

#### 📍 **地理位置**
```
地理位置：位于未来科技城核心区，毗邻阿里巴巴西溪园区...
```

#### 👥 **核心客流**（三类客群）
```
企业员工：周边阿里、网易等大厂，年轻员工5万+...
高校学生：杭师大距离2公里，学生3万人...
商旅客群：西湖会展中心1公里，商务出差频繁...
```
**别名支持**：
- "企业员工" / "企业年轻员工" / "企业"
- "高校学生" / "学生" / "大学生"
- "商旅客群" / "商务客群" / "商旅"

#### 🏢 **竞争格局**（三个维度）
```
直接竞品：3km内无电竞酒店，市场空白
潜在竞品：2家传统网咖，装修陈旧
替代娱乐：电影院2家，KTV 3家
```

#### 🎮 **电竞馆分布**
```
1km以内电竞馆：5家
1-2km电竞馆：8家
2-3km电竞馆：3家
电竞馆备注：市场需求旺盛但不饱和
```
**别名支持**：
- "1km以内" / "1公里以内"
- "1-2km" / "1到2公里"

#### 🏨 **电竞酒店分布**（动态列表）
```
电竞酒店1：XX电竞酒店,1.2公里,50间房
电竞酒店2：YY主题酒店,2.8公里,30间房
```
**格式要求**：`[名称],[距离],[房间数]`（用逗号分隔）

#### 🏢 **商务酒店分布**（动态列表）
```
商务酒店1：如家酒店,0.8公里,80间房
商务酒店2：汉庭酒店,1.5公里,100间房
商务酒店3：全季酒店,2.0公里,120间房
```

## 🚀 使用步骤

### 第一步：准备文本
按照上述格式整理项目信息文本，每个字段独立一行。

### 第二步：粘贴文本
在"智能解析文本"输入框中粘贴准备好的文本。

### 第三步：点击解析
点击"一键解析并填充表单"按钮，系统自动识别并填充。

### 第四步：检查表单
系统会自动滚动到表单区域，检查填充的数据是否正确。

### 第五步：补充调整
如有缺失或错误，可手动在表单中修改。

### 第六步：提交评估
点击"提交评估"按钮，获取智能评估结果。

## 💡 使用技巧

### ✅ 推荐做法

1. **字段完整**：尽量包含所有6大核心字段
2. **格式规范**：使用冒号分隔字段名和内容
3. **数据准确**：酒店列表使用逗号分隔三要素
4. **单位统一**：距离用"公里"或"km"，房间数用"间房"

### ⚠️ 常见问题

**Q1: 解析后部分字段为空？**
- 检查字段名称是否正确（如"企业员工"不要写成"企业客户"）
- 确认使用了冒号（：或:）分隔
- 每个字段独立一行

**Q2: 酒店列表没有解析成功？**
- 确保格式为：`电竞酒店1：名称,距离,房间数`
- 三个要素用逗号（,或，）分隔
- 数字可以带单位，系统会自动去除

**Q3: 电竞馆数量解析错误？**
- 确保格式为：`1km以内电竞馆：5家`
- 数字后可以加"家"，系统会自动识别

**Q4: 解析失败怎么办？**
- 检查文本格式是否符合规范
- 尝试手动填写部分字段
- 查看浏览器控制台的错误信息

## 🔧 技术实现

### 解析引擎
- **正则表达式匹配**：识别字段名称和内容
- **智能分词**：自动分离数字、单位、文本
- **动态列表构建**：自动创建酒店条目表单

### 数据流
```
用户粘贴文本 
  ↓
parseInvestmentText(text) 解析为结构化数据
  ↓
fillFormWithParsedData(data) 填充表单字段
  ↓
saveAssessmentDraft() 自动保存草稿
  ↓
用户提交评估
```

### 支持的字段匹配模式
```javascript
// 地理位置
/^地理位置[：:]/i

// 企业员工
/^企业[员工年轻]*[：:]/i || /企业.*员工/i

// 高校学生
/^[高校大学]*学生[：:]/i || /高校.*学生/i

// 商旅客群
/^商旅|商务.*[客群旅客][：:]/i

// 电竞馆
/1\s*km\s*以内|1\s*公里以内/i
/1\s*-\s*2\s*km|1\s*到\s*2\s*公里/i

// 酒店列表
/电竞酒店\d+[：:]/i
/商务酒店\d+[：:]/i
```
