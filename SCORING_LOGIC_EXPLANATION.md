# 项目投资评估系统 - 评分逻辑详解

## 📊 总体评分架构

### 综合得分计算公式
```
综合得分 = 优势得分(60%) + 风险得分(40%)
```

### 优势得分 (60%权重)
```
优势得分 = (地理位置 × 40% + 核心客流 × 60%) × 60%
```

### 风险得分 (40%权重)
```
风险得分 = [竞争格局 × 30% + 竞品分布 × 70%] × 40%

其中竞品分布 = 电竞馆 × 20% + 电竞酒店 × 40% + 商务酒店 × 10%
```

---

## 🎯 六大维度详细评分规则

### 1. 地理位置 (0-10分)

**评分逻辑：基础分5分，根据关键词加减分**

| 关键词类别 | 关键词示例 | 影响 |
|---------|----------|-----|
| **高价值** (+1.5分/个) | 阿里巴巴、杭师大、未来科技城、核心区、商圈、CBD、高新区 | 每个+1.5 |
| **中等价值** (+1分/个) | 产业园、大学城、地铁站、商业区、交通枢纽 | 每个+1.0 |
| **低价值** (-2分/个) | 郊区、偏远 | 每个-2.0 |

**示例**：
- 描述"阿里巴巴旁边，杭师大附近" → 5 + 1.5 + 1.5 = **8分**
- 描述"郊区工业园" → 5 + 1 - 2 = **4分**

**设计理念**：地理位置是电竞酒店成功的基础，优质地理位置带来稳定客流。

---

### 2. 核心客流 (0-10分)

**评分逻辑：三类客群加权平均**

| 客群类型 | 权重 | 评分标准 |
|---------|-----|---------|
| 企业年轻员工 | 40% | 描述长度 + 关键词 |
| 高校学生 | 35% | 描述长度 + 关键词 |
| 商旅与参会客群 | 25% | 描述长度 + 关键词 |

**单项客群评分**：
- 描述 < 20字 → 0分
- 描述 20-50字 → 6分
- 描述 50-100字 → 8分
- 描述 > 100字 → 9分

**关键词加分** (每个+0.5分)：
- 密集、活跃、需求旺盛、消费能力强、目标客群

**计算示例**：
```
企业员工描述80字含"需求旺盛" → (8 + 0.5) × 40% = 3.4
高校学生描述30字 → 6 × 35% = 2.1
商旅客群描述15字 → 0 × 25% = 0

总分 = 3.4 + 2.1 + 0 = 5.5分
```

**设计理念**：电竞酒店的核心客户是年轻群体，三类客群覆盖越全面，市场基础越牢固。

---

### 3. 竞争格局 (0-10分)

**评分逻辑：基础分10分（假设无竞争），根据竞品情况扣分**

| 竞品类型 | 关键词 | 扣分 |
|---------|-------|-----|
| **直接竞品** | 无/较少 | 0分 |
| | 一般 | -2分 |
| | 较多/密集 | -4分 |
| **潜在竞品** | 较多 | -2分 |
| | 一般 | -1分 |
| **替代娱乐** | 丰富/较多 | -1.5分 |

**示例**：
- "直接竞品较多，潜在竞品一般" → 10 - 4 - 1 = **5分**
- "直接竞品较少，无替代娱乐" → 10 - 0 = **10分**

**设计理念**：⚠️ **竞争越激烈，风险越大，得分越低！** 这是风险维度的核心逻辑。

---

### 4. 电竞馆分布 (0-10分)

**评分逻辑：3KM范围内电竞馆数量决定竞争密度**

```
3KM总数 = 1km以内 + 1-2km + 2-3km
```

| 3KM内电竞馆总数 | 得分 | 市场状态 |
|----------------|------|---------|
| ≤ 15家 | 9.5分 | 竞争适中，市场健康 |
| 16-20家 | 7.5分 | 竞争较激烈 |
| 21-25家 | 6.0分 | 竞争激烈 |
| 26-30家 | 4.0分 | 竞争过度 |
| > 30家 | 2.0分 | 市场饱和 |

**设计理念**：
- ✅ **适量的电竞馆（≤15家）说明市场有需求但不饱和**
- ⚠️ **过多电竞馆（>25家）说明市场竞争激烈，风险大**

---

### 5. 电竞酒店分布 (0-10分) ⭐核心风险指标

**评分逻辑：根据距离和房间数计算竞争强度**

```python
竞争强度 = Σ (房间数 × 距离权重)

距离权重：
- ≤1km: 0.8 (影响最大)
- 1-2km: 0.5
- 2-3km: 0.3
```

| 竞争强度得分 | 项目得分 | 市场状态 |
|------------|---------|---------|
| ≤ 30 | 10.0分 | 市场空白，先发优势 |
| 31-60 | 8.0分 | 竞争较小 |
| 61-100 | 6.0分 | 竞争一般 |
| > 100 | 4.0分 | 竞争激烈 |
| 无电竞酒店 | 10.0分 | 完全空白市场 |

**计算示例**：
```
A酒店：距离0.5km，50间房 → 50 × 0.8 = 40
B酒店：距离1.5km，30间房 → 30 × 0.5 = 15
竞争强度 = 40 + 15 = 55 → 得分8.0分
```

**设计理念**：
- ⚠️ **电竞酒店是直接竞争对手，越多风险越大**
- ✅ **无电竞酒店 = 市场空白 = 最高得分10分**

---

### 6. 同档次商务酒店分布 (0-10分)

**评分逻辑：商务酒店既带来客流，又带来竞争**

```
总房间数 = Σ 各商务酒店房间数
```

| 3KM内总房间数 | 得分 | 市场判断 |
|-------------|------|---------|
| 0间 | 6.0分 | 无商务氛围，缺少客流支撑 |
| 1-100间 | 7.0分 | 商务氛围一般 |
| 101-300间 | 9.0分 | ✅ 商务氛围良好（最佳） |
| 301-500间 | 8.0分 | 竞争开始增大 |
| > 500间 | 6.0分 | 竞争激烈 |

**设计理念**：
- ✅ **适量商务酒店（100-300间）说明商务客流稳定，有利于电竞酒店发展**
- ⚠️ **过多商务酒店（>500间）会抢走部分客源**

---

## 🎯 核心问题回答

### ❓ "周围竞争越多，风险越大吗？"

**答：是的！这是评估系统的核心逻辑。**

| 竞争类型 | 影响程度 | 评分逻辑 |
|---------|---------|---------|
| **电竞酒店** (直接竞争) | ⚠️⚠️⚠️ 最高 | 越多得分越低 (10→4分) |
| **竞争格局** (市场竞品) | ⚠️⚠️ 高 | 越多得分越低 (10→5分) |
| **电竞馆** (市场需求) | ⚠️ 中 | 过多说明竞争激烈 (9.5→2分) |
| **商务酒店** (辅助竞争) | ⚠️ 低 | 适量最好，过多有风险 |

## 💡 实际案例分析

### 案例1：高价值项目 (综合得分: 8.7)

**输入数据**：
- 地理位置：阿里巴巴总部旁，未来科技城核心区 → **9.5分**
- 核心客流：企业员工密集，高校学生活跃 → **8.5分**
- 竞争格局：直接竞品较少 → **8.0分**
- 电竞馆分布：3KM内10家 → **9.5分**
- 电竞酒店分布：无 → **10.0分** ⭐关键！
- 商务酒店：3KM内200间 → **9.0分**

**计算过程**：
```
优势得分 = (9.5 × 0.4 + 8.5 × 0.6) × 0.6 = 5.34
风险得分 = [8.0 × 0.3 + (9.5×0.2 + 10×0.4 + 9×0.1)×0.7] × 0.4 = 3.37
综合得分 = 5.34 + 3.37 = 8.71 → 高投资价值
```

---

### 案例2：低价值项目 (综合得分: 4.8)

**输入数据**：
- 地理位置：郊区偏远位置 → **3.0分**
- 核心客流：客群覆盖不足 → **4.0分**
- 竞争格局：直接竞品密集 → **5.0分**
- 电竞馆分布：3KM内35家 → **2.0分** ⚠️过度饱和
- 电竞酒店分布：3家，总房间数150间 → **4.0分** ⚠️竞争激烈
- 商务酒店：3KM内600间 → **6.0分**

**计算过程**：
```
优势得分 = (3.0 × 0.4 + 4.0 × 0.6) × 0.6 = 2.16
风险得分 = [5.0 × 0.3 + (2.0×0.2 + 4.0×0.4 + 6.0×0.1)×0.7] × 0.4 = 1.54
综合得分 = 2.16 + 1.54 = 3.70 → 投资价值偏低
```

## 🎯 关键洞察

### ✅ 什么样的项目适合投资？

1. **地理位置优越**（阿里、杭师大等核心区域）
2. **目标客群明确且活跃**（企业员工 + 高校学生）
3. **电竞酒店市场空白**（3KM内无或很少）⭐最重要
4. **电竞馆数量适中**（≤15家，说明有需求但不饱和）
5. **商务酒店适量**（100-300间，提供客流支撑）
6. **竞争格局良好**（直接竞品少）

### ⚠️ 什么样的项目风险高？

1. **地理位置偏远**（郊区、交通不便）
2. **目标客群不明确**（客流描述模糊）
3. **电竞酒店密集**（3KM内3家以上）⭐风险最大
4. **电竞馆过度饱和**（>25家，市场竞争激烈）
5. **直接竞品众多**（竞争格局复杂）

## 📌 总结

**评分逻辑的核心思想：**

1. **优势维度（60%）**：看市场机会
   - 地理位置 + 核心客流 → 决定天花板

2. **风险维度（40%）**：看竞争压力
   - 竞争格局 + 竞品分布 → 决定成功概率
   - ⚠️ **竞争越多，风险越大，得分越低**

3. **平衡原则**：
   - 电竞馆太少 = 市场需求不足
   - 电竞馆太多 = 市场过度竞争
   - 电竞酒店越少越好 = 市场空白机会
   - 商务酒店适量最好 = 提供客流但不过度竞争
