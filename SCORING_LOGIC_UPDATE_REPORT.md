# 竞争格局评分逻辑更新报告

## 📊 更新内容

### 问题描述
用户反馈："竞争格局"得分偏高（8.0分），未充分反映实际竞争状况。

### 根本原因
1. **关键词检测过于简单**：只检测"较多"、"密集"等固定词汇
2. **未分析实际数量**：忽略了电竞酒店、商务酒店、电竞馆的真实数量
3. **潜在竞品影响低估**：商务酒店可能转型，但未扣分
4. **替代娱乐影响低估**：网咖等替代选择未被充分考虑

---

## ✅ 解决方案

### 新评分逻辑：结合数量数据的精确计算

```python
def calculate_competitive_pattern_score(self) -> float:
    """
    计算竞争格局得分 (0-10分)
    
    评分规则:
    - 结合文本描述和实际竞品数量
    - 直接竞品（电竞酒店）影响最大
    - 潜在竞品（商务酒店）和电竞馆密度影响次之
    - 竞争对手越多，得分越低，风险越高
    """
    pattern = self.data.get('competitive_pattern', {})
    
    # 获取实际竞品数据
    esports_hotels = self.data.get('esports_hotel_distribution', [])
    business_hotels = self.data.get('business_hotel_distribution', [])
    venue_dist = self.data.get('esports_venue_distribution', {})
    
    # 统计数量
    esports_count = len(esports_hotels)
    business_count = len(business_hotels)
    venue_count = venue_dist.get('1km以内', 0) + venue_dist.get('1-2km', 0) + venue_dist.get('2-3km', 0)
    
    score = 10.0
    
    # 1. 电竞酒店竞争（权重最高）
    if esports_count == 0:
        score -= 0  # 市场空白
    elif esports_count <= 2:
        score -= 0.5  # 竞争很少
    elif esports_count <= 4:
        score -= 1.5  # 竞争适中
    elif esports_count <= 6:
        score -= 2.5  # 竞争较激烈
    elif esports_count <= 8:
        score -= 3.5  # 竞争激烈
    else:
        score -= 4.5  # 竞争非常激烈
    
    # 2. 电竞馆密度（次要影响）
    if venue_count >= 30:
        score -= 1.5  # 市场过度饱和
    elif venue_count >= 25:
        score -= 1.0
    elif venue_count >= 20:
        score -= 0.5
    
    # 3. 商务酒店潜在竞争
    if business_count >= 10:
        score -= 1.5
    elif business_count >= 8:
        score -= 1.0
    elif business_count >= 5:
        score -= 0.5
    
    # 4. 文本描述修正（加分项）
    direct = pattern.get('直接竞品', '')
    if '市场空白' in direct or '定位差异' in direct or '差异化' in direct:
        score += 0.5  # 虽有竞品但存在差异化机会
    
    return max(0, min(10, score))
```

## 🎯 评分逻辑全景

### 核心问题回答

**Q1: 周边竞品越多，是否风险越高？**
**A: 是的！** 评分系统明确设计为：

| 竞品类型 | 数量增加 | 得分变化 | 风险变化 |
|---------|---------|---------|---------|
| 电竞酒店 | 0 → 9+家 | 10分 → 5.5分 | 低 → 高 |
| 电竞馆 | 0 → 30+家 | 9.5分 → 2分 | 低 → 高 |
| 商务酒店 | 0 → 10+家 | 额外-0到-1.5分 | 低 → 中 |

### 各维度评分规则总结

#### 1. 地理位置（0-10分）
- **基础分**：5.0分
- **高价值关键词**（阿里/杭师大/核心区）：每个+1.5分
- **中等价值关键词**（产业园/大学城）：每个+1.0分
- **低价值关键词**（郊区/偏远）：每个-2.0分

#### 2. 核心客流（0-10分）
**加权平均**：企业员工40% + 高校学生35% + 商旅25%

**每个客群评分**：
- 描述 < 20字符 → 0分
- 描述 20-50字符 → 6分
- 描述 50-100字符 → 8分
- 描述 > 100字符 → 9分
- 每个正向关键词 → +0.5分

#### 3. 竞争格局（0-10分）⭐ 本次更新
**基础分**：10.0分

**扣分规则**：
- 电竞酒店0家 → -0分
- 电竞酒店1-2家 → -0.5分
- 电竞酒店3-4家 → -1.5分
- 电竞酒店5-6家 → -2.5分 ⬅️ **你的情况**
- 电竞酒店7-8家 → -3.5分
- 电竞酒店9+家 → -4.5分

**次要影响**：
- 电竞馆20-25家 → -0.5分
- 电竞馆25-30家 → -1.0分
- 电竞馆30+家 → -1.5分
- 商务酒店5-8家 → -0.5分
- 商务酒店8-10家 → -1.0分
- 商务酒店10+家 → -1.5分

**加分项**：
- 文本提到"市场空白"/"定位差异" → +0.5分

#### 4. 电竞馆分布（0-10分）
- ≤15家 → 9.5分
- 16-20家 → 7.5分
- 21-25家 → 6.0分 ⬅️ **你的23家**
- 26-30家 → 4.0分
- 31+家 → 2.0分

#### 5. 电竞酒店分布（0-10分）
**竞争强度计算**：
```
竞争强度 = Σ(房间数 × 距离权重)
- 距离 ≤ 1km → 权重0.8
- 距离 1-2km → 权重0.5
- 距离 > 2km → 权重0.3
```

**评分规则**：
- 竞争强度 ≤ 30 → 10分
- 竞争强度 31-60 → 8分
- 竞争强度 61-100 → 6分 ⬅️ **你的52分**
- 竞争强度 > 100 → 4分

#### 6. 商务酒店分布（0-10分）
- 0家 → 6.0分（缺少客流支撑）
- ≤100间 → 7.0分
- 101-300间 → 9.0分
- 301-500间 → 8.0分
- 500+间 → 6.0分（竞争激烈）

## 🧮 综合得分计算公式

```
优势得分 = (地理位置 × 40% + 核心客流 × 60%) × 60%
风险得分 = (竞争格局 × 30% + 电竞馆 × 20% + 电竞酒店 × 40% + 商务酒店 × 10%) × 40%
综合得分 = 优势得分 + 风险得分
```

**价值等级判定**：
- ≥ 8.5分 → 高投资价值（绿色）
- 7.0-8.5分 → 较高投资价值（蓝色）
- 5.5-7.0分 → 中等投资价值（黄色）
- < 5.5分 → 投资价值偏低（红色）
