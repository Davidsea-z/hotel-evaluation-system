```

IRR = MIRR（考虑复投，再投资率 = 预期年收益率）
```

## 📝 使用说明

### 用户操作
1. 打开投资模型计算器
2. 查看"YITO期限（按18%年收益率）"卡片
   - 大数字：天数（如：1002）
   - 小字：月数（如：33.4月）
3. 查看"IRR（考虑复投）"卡片
   - 标签固定为"IRR（考虑复投）"
   - 无论选择哪种分账频率
4. 调整"分账频率"下拉框
   - IRR 标签保持不变
   - 计算结果会相应调整

### 分账频率说明
- **日分账：** 每天回收现金流，按预期收益率复投
- **周分账：** 每周回收现金流，按预期收益率复投
- **双周分账：** 每两周回收现金流，按预期收益率复投
