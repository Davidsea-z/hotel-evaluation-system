# 酒店电竞化改造项目评估系统使用指南

## 📋 功能概述

**项目评估系统**是一个智能化的投资决策辅助工具，通过输入酒店项目的核心数据，自动评估项目投资价值，提供综合评分、详细指标分析和投资建议。

---

## 🎯 核心功能

### 1. 智能评分算法（100分制）

评估系统基于四大维度进行综合打分：

| 维度 | 权重 | 评分要素 |
|------|------|----------|
| **财务收益** | 30分 | 预计年ROI（投资回报率） |
| **市场潜力** | 25分 | 区域位置评级 |
| **回本速度** | 25分 | 回本周期（月） |
| **风险控制** | 20分 | 设备配置、预算合理性 |

### 2. 评级体系

根据综合得分自动判定项目等级：

- **85-100分：优秀项目** 🟢 - 强烈推荐投资
- **70-84分：良好项目** 🔵 - 推荐投资
- **55-69分：中等项目** 🟡 - 谨慎投资，需优化
- **0-54分：需改进** 🔴 - 不建议投资

### 3. 详细指标分析

系统自动计算并展示：

**财务指标**：
- 预计年营收（万元）
- 预计年利润（万元）
- 投资回报率 ROI（%）

**市场指标**：
- 区域评级（核心商圈/开发区/融合区/景区）
- 房价提升潜力（%）
- 入住率提升空间（%）

**回本分析**：
- 回本周期（月）
- 回本速度评级（极快/较快/中等/较慢/慢）

**风险评估**：
- 设备配置评级（高端/中端/入门）
- 预算合理性判断
- 风险等级（低/中低/中等/较高）

### 4. 智能投资建议

系统根据评估结果自动生成：

- **综合评价**：项目整体投资价值判断
- **优势分析**：列举项目核心竞争力（2-4条）
- **风险提示**：识别潜在风险点（1-5条）
- **优化建议**：提供具体改进方案（4-6条）

### 5. 对标案例展示

自动匹配3个相似的真实改造案例：
- 浙江人民广场店（核心商圈，ROI 275%）
- 华东全新电竞店（城市开发区，ROI 88%）
- 华东商务酒店（城市开发区，ROI 257%）
- 华东存量酒店（产城融合区，ROI 90%）

## 💡 评分规则详解

### 1. 财务收益评分（30分）

根据预计年ROI自动评分：

- **ROI ≥ 200%**：满分 30分（优秀）
- **ROI ≥ 150%**：25分（良好）
- **ROI ≥ 100%**：20分（中等）
- **ROI ≥ 80%**：15分（一般）
- **ROI < 80%**：10分（较差）

**计算公式**：
```
预计年营收 = 改造房间数 × 目标房价 × (目标入住率/100) × 30天 × 12月 / 10000
预计年利润 = 预计年营收 × 65%（行业利润率）
ROI = (预计年利润 / 改造总预算) × 100%
```

### 2. 市场潜力评分（25分）

根据项目位置自动评分：

- **核心商圈**：25分（房价提升1.5倍，入住率+35%）
- **景区周边**：22分（房价提升1.4倍，入住率+30%）
- **城市开发区**：20分（房价提升1.35倍，入住率+30%）
- **产城融合区**：18分（房价提升1.25倍，入住率+25%）

### 3. 回本速度评分（25分）

根据回本周期自动评分：

- **≤ 6个月**：25分（极快回本）
- **6-10个月**：20分（较快回本）
- **10-15个月**：15分（中等速度）
- **15-24个月**：10分（较慢回本）
- **> 24个月**：5分（慢速回本）

**计算公式**：
```
回本周期（月）= (改造总预算 / 预计年利润) × 12
```

### 4. 风险控制评分（20分）

综合考察三个维度：

#### a) 显卡配置（基础分）
- **高端（RTX5070+）**：20分
- **中端（RTX5060Ti/5070）**：18分
- **入门（RTX5060）**：15分

#### b) 预算合理性（扣分项）
```
单房预算 = 改造总预算 × 10000 / 改造房间数
```
- **2.5-3.5万元/间**：合理（不扣分）
- **< 2.5万元/间**：偏低（扣2分）
- **> 3.5万元/间**：偏高（扣1分）

#### c) 设备占比（扣分项）
- **70-80%**：优秀（不扣分）
- **60-70%**：良好（扣2分）
- **< 60%**：不足（扣4分）

## ⚠️ 注意事项

### 1. 数据准确性
- 确保输入数据真实准确，特别是改造前数据
- 目标房价和入住率应基于市场调研，不可过度乐观
- 预算应包含电竞设备、软装、弱电等全部成本

### 2. 参考标准
- **电竞设备占比**：建议75%以上，确保硬件竞争力
- **单房预算**：建议2.5-3.5万元，保证改造质量
- **目标入住率**：建议85%，符合行业平均水平
- **显卡配置**：建议RTX5060Ti及以上，确保3-5年不落后

### 3. 评估局限性
- 本系统基于历史数据和行业标准建模
- 实际收益受市场环境、运营能力等多因素影响
- 建议结合实地考察、市场调研综合决策

### 4. 优化建议
系统会根据具体情况提供优化建议：
- 房价定位是否合理
- 入住率目标是否可达
- 设备预算是否充足
- 改造规模是否适当

```javascript
// 财务收益评分（30分）
if (roi >= 200) score = 30;
else if (roi >= 150) score = 25;
else if (roi >= 100) score = 20;
else if (roi >= 80) score = 15;
else score = 10;

// 市场潜力评分（25分）
score = locationRatings[location].score;

// 回本速度评分（25分）
if (paybackMonths <= 6) score = 25;
else if (paybackMonths <= 10) score = 20;
else if (paybackMonths <= 15) score = 15;
else if (paybackMonths <= 24) score = 10;
else score = 5;

// 风险控制评分（20分）
score = gpuRatings[gpuLevel].score;
// 根据预算和设备占比调整
```
